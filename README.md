## ğŸš€ Getting Started

### Prerequisites

- Node.js (v16 or higher)
- npm or yarn
- Git

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/WA-code-Github-Account/humanoid-robotics-book.git
   cd humanoid-robotics-book/docs-site
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start development server**
   ```bash
   npm start
   ```

   The site will open at `http://localhost:3000`

### Building for Production

```bash
npm run build
```

This generates static content into the `build` directory.

### Deployment

```bash
npm run deploy
```

Or manually:
```bash
cd build
git init
git add -A
git commit -m "Deploy to GitHub Pages"
git branch -M gh-pages
git remote add origin https://github.com/WA-code-Github-Account/humanoid-robotics-book.git
git push -f origin gh-pages
```
